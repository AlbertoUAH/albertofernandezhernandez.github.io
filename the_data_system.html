The Data System

I know there are thousands of blogs about Data Science, so I want to make something "a little bit different"

Following the crips-dm methodology, I'll post in several themes including:


  • Data Understanding: collecting initial data, data description & exploration
  • Data Preparation: data selection, cleaning, filtering, construction & formatting
  • Modeling: generating test design and building models
  • Evaluation: Assestment of Data Mining results
  • Deployment: APIs for deployment, monitoring & maintenance plan

Enjoy! #DataDataAndData
